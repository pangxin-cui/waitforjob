一、引言

盘盘工作室（个人开发者，以下简称"我们"）是移动应用程序"待业人生"（以下简称"本游戏"）的开发者。我们高度重视用户（以下简称"您"）的个人信息和隐私。

本游戏为一款免费的单机文字卡牌游戏，无需注册账号、无需实名认证、不包含内购。我们承诺本游戏数据仅存储在您的设备本地，不会上传至任何服务器。AI 叙事增强功能为可选模块，由用户自行配置第三方 API Key 后启用。

二、我们如何收集和使用您的个人信息

（一）游戏基本信息

本游戏无需注册账号，运行时不要求您提供姓名、手机号码、电子邮箱或其他身份信息。您启动游戏即可直接开始游玩。

（二）本地存档数据

您在本游戏中的游玩记录（包括但不限于游戏进度、成就、卡牌收集、设置偏好等）全部存储在您设备的本地存储（localStorage）中。我们不会也无技术能力将这些数据上传或同步至任何服务器。

（三）AI 叙事增强功能（可选）

如您在"设置"中自行启用 AI 叙事增强功能，您需要自行提供第三方 AI 服务商的 API Key（如 OpenAI、DeepSeek、豆包等）。该 API Key 仅保存在您设备的本地存储中，本游戏不会读取、收集或上传您的 API Key。AI 调用产生的数据交互直接发生在您的设备与对应的 AI 服务商之间，本游戏不参与数据的中转或存储。

启用 AI 功能后，本游戏需要网络连接以发送 API 请求；未启用时，本游戏可完全离线运行。

（四）设备权限

本游戏仅在您启用 AI 叙事增强功能时申请网络连接权限。您有权随时在设备设置中关闭网络权限。关闭后不影响游戏的核心玩法。

本游戏不使用麦克风、摄像头、位置信息、通讯录等权限。本游戏不集成任何第三方广告 SDK、分析工具或用户行为追踪技术。

三、我们如何存储您的个人信息

本游戏不拥有服务器，所有数据仅存储在您的设备本地。如您卸载本游戏或清除应用数据，您的本地存档将被永久删除且无法恢复。

我们不对您的本地数据做远程备份。建议您定期使用游戏内的"存档"功能导出数据文件自行保管。

四、我们如何共享、转让、公开披露您的个人信息

由于本游戏不收集您的个人信息，我们不存在共享、转让或公开披露您个人信息的情形。

如您在游戏中启用 AI 叙事增强功能，您直接向第三方 AI 服务商发送的数据受该服务商的隐私政策约束，与本游戏无关。

五、我们如何保护您的个人信息

由于您的游戏数据全部存储在本地设备中，您对自身数据安全负有最终责任。建议您为设备设置锁定密码、谨慎安装来源不明的应用程序，以降低本地数据泄露的风险。

六、您的权利

由于本游戏不收集您的个人信息、不设立用户账户，您可以随时通过卸载应用来终止服务。您有权通过设备设置管理网络权限。

七、未成年人保护

本游戏适龄提示为 12+，游戏内容不包含色情、暴力、赌博等不适宜未成年人的内容。本游戏不面向 12 岁以下儿童设计，不收集任何儿童的个人信息。

八、本隐私政策如何更新

本隐私政策可能会适时更新。重大变更将以游戏内弹窗通知的形式告知。继续使用本游戏即视为您同意更新后的隐私政策。

九、如何联系我们

如您有隐私相关问题或建议，您可以通过 TapTap 开发者后台工单系统联系：在"待业人生"游戏页面底部，点击"联系开发者"。

我们将在收到反馈后的十五个工作日内回复。
